# Grotthuss_paper

Code and simulation files accompanying our paper on Grotthuss proton transport, modeled using the REACTER protocol in LAMMPS.

## Repository structure

```
Grotthuss_paper/
├── REACTER_polymer_simulations/     # REACTER simulations of the polymer system
│   ├── input_files/                 # LAMMPS data/input scripts, force field files
│   ├── scripts/                     # Setup, analysis, 
│
└── REACTER_bulkwater_simulations/   # REACTER simulations of the bulk water system
    ├── input_files/                 # LAMMPS data/input scripts, force field files
    ├── scripts/                     # Setup, analysis, and post-processing scripts
   
```
